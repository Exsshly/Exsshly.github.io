装机配置

Date: 2023-06-20  14:00:12

现在最烦的就是重装系统了，不知道从什么时候开始的。

高二高三时却对此乐此不疲，班里的电脑最多同时存在了三个系统并且全盘无冲……

趁着两次卸载掉三哥牌的Chromium，那每个月的累计更新就是更新不上，0x800f081f，错误代码我都会背了，从三月卸载掉Edge到杭州的累计更新更不上，一气之下屏蔽掉更新回去Win10，再后来装22H1，我以为就万事大吉，结果还是不行。后来在哪看到那个错误代码是缺少什么🐦组件，思来想去我只卸掉的所谓不能卸载的三哥牌Chromium。今天无事就再装一遍。

本着前人栽树后人乘凉的原则，其实我早就想写一个清单，每次装完系统手忙脚乱，为了以后能够流畅些(是TMD真的烦装系统)，这次干脆就把个人操作的流程记录一下，后面就不用东拼西凑了，直接看这里就好。

高级格式化前

  • 备份C盘用户目录下各种文件(包括不限于微信聊天记录以及git配置等)

  • 备份控制面板卸载程序页所有程序

  • 备份桌面及磁贴(W11已经取消这里延续叫法)布局

  • 有一个目标系统的镜像(必要时有一个Office镜像以及MS运行库)

装机

  • 这里不写，爱怎么装怎么装

开机后设置

  • 过oobe(这里不想签字登录邮箱就写1@1.com密码随便)
  • 开机卸载应用，有时候可能需要卸载两次(重启之后又rm回来了！)
    以下都在W1x系设置里配置
  • 设置屏幕缩放125%(个人习惯以及显示器问题)
  • 杀掉麦克风(从来没用过)
  • 设置屏幕和睡眠
  • 开启剪贴板历史记录
  • 设置鼠标指针
  • 设置触摸板
  • 配置开始页以及任务栏
  • 设置开机PIN

其他

下面是个人习惯，不想系统保留空间太大且不用的情况下释放掉空间，都需要管理员权限运行

  • 关闭休眠 
    通过powercfg -h off关闭，之后还要通过电源设置里关闭快速启动以及睡眠
  • 关闭系统还原(似乎默认就是关闭状态，这里保险起见个人还是去控制面板里确认一下)
  • 关闭保留空间

dism.exe /Online /Set-ReservedStorageState /State:Disabled

reg add "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\ReserveManager" /f /v "BaseHardReserveSize" /t REG_QWord /d "0" >nul

reg add "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\ReserveManager" /f /v "BaseSoftReserveSize" /t REG_QWord /d "0" >nul

reg add "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\ReserveManager" /f /v "HardReserveAdjustment" /t REG_QWord /d "0" >nul

reg add "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\ReserveManager" /f /v "MinDiskSize" /t REG_QWord /d "0" >nul

reg add "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\ReserveManager" /f /v "ShippedWithReserves" /t REG_DWORD /d "0" >nul

  • 装上Office三件套
  • 使用Dism++调教其他非外显设置
  • 根据备份单装回软件，必要时恢复环境
    装translucentTB，W11还要额外配置个什么东西才透明啊md，参https://www.bilibili.com/read/cv22300526
    开QuickLook，开WallpaperEngine(可选，22H2上只要开了就掉帧什么意思)
    还原磁贴，还原经典记事本、画图以及创建Aria
    装Hexo:npm install -g hexo-cli
    还原C盘用户目录下各种文件(包括不限于微信聊天记录以及git配置等)
    WindowsUpdate安装更新
    Kclener调教一遍
    WiseCare调教两遍
    CCClener调教三遍
    如是……最想想起什么再补充什么，或者什么时候装系统觉得不合适再增删……
    PS：一个月内装了三次系统才想起来写，惭愧……
